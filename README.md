# Enterprise-Campus-Network-Design-HQ-Branch-Simulation-Project-
I designed and simulated a secure, scalable campus network connecting a Head Office and Branch campus with high availability, segmentation, and security best practices.
The solution focuses on scalability, redundancy, and security using Cisco technologies.

## Technologies Used
- Cisco ASA Firewall
- OSPF Routing
- VLAN Segmentation
- HSRP Gateway Redundancy
- EtherChannel (LACP)
- Wireless LAN Controller (WLC)
- DHCP Services
- IPsec Site-to-Site VPN
- STP PortFast and BPDU Guard
- Access Control Lists (ACL)

## Key Features
- Site-to-site IPsec VPN between campuses
- Centralized DHCP servers
- VLAN segmentation for security
- ASA firewall security zones
- High availability using HSRP
- Inter-VLAN routing on multilayer switches
- Core Layer (Catalyst 3850)
- Distribution Layer
- Access Layer (Catalyst 2960)
